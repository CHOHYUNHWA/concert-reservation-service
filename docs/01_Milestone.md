# Milestone

> **참고 사항**
> - **1MD = 3MH**

---

2. **요구사항 분석** (1MD)
3. **시퀀스 다이어그램 작성** (1MD)
4. **ERD 설계** (1MD)
5. **API 명세서 작성** (1MD)
6. **Mock API 개발** (0.5MD)
